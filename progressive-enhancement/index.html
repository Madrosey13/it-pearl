Progressive Enhancement - Madison Young

Progressive enhancement is a design philosophy that can be described as providing functionality and accessibility to a website's content. It also tries to deliver the best possible experience to users with modern browsers that can run the code that the website is using.

The use of progressive enhancement may sound irrelavent to modern web design, but it is very important when designing a site that is accessible to all people, inlcuding ones with disabilities. Here are some reasons why progressive enhancement is important:

  • Websites that are built with progressive enchancement tend to load faster since they prioritize essential content and functionality
  • Progressive enhancement ensures that your site still remains functional and accessible as new technologies emerge
  • Progressive enhancement provides a tailored user experience, which means that users that have basic browsers still have a functional and readable site, and the users that have advanced browsers can enjoy enhanced features that the users with the basic browser could not
